# currencyapp

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Build docker
```
docker build -t currencyapp .
```

### Run docker
```
docker run -it -p 8080:8080 --rm --name currencyapp1 currencyapp
```
